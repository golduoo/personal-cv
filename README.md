# Personal CV Website (Assignment 524)

本项目用于完成 `Final Individual Project — Personal Job-Seeking Website` 作业。

## 文件说明

- `index.html`：网站主页面（已包含作业要求的 6 个模块）
- `styles.css`：网站样式
- `524_Individual_Project_总结.md`：作业要求中文总结

## 你现在要做的 4 件事

1. 打开 `index.html`，把所有“待补充/占位符”替换成你的真实信息：
   - 学校、专业、毕业时间
   - 邮箱、LinkedIn、项目链接、简历链接
   - 两个项目内容按你的真实经历修改
2. 确认你的简历 PDF 可访问（建议放到仓库并在页面引用）。
3. 推送到 GitHub 仓库 `personal-cv`。
4. 在仓库设置里开启 GitHub Pages。

## GitHub Pages 发布步骤

1. 把当前目录文件上传到仓库根目录（至少包含 `index.html`）。
2. GitHub 仓库页面进入 `Settings` → `Pages`。
3. `Build and deployment` 里选择：
   - Source: `Deploy from a branch`
   - Branch: `main`（或你的默认分支） / `/ (root)`
4. 保存后等待 1-3 分钟，访问生成的网址。
